# UI Designer Tools

## Design Resources
- **read_file** — 阅读设计系统文档、品牌指南、组件规格
- **write_file** — 创建设计说明文档、交互规格、组件文档
- **web_fetch** — 获取设计灵感、竞品截图、设计趋势

## Knowledge
- **search_knowledge** — 查询设计系统、组件库文档、设计决策记录
- **memory_ops** — 记录设计决策、用户反馈、迭代历史

## Workflow
- **skill_load** — 加载 planning 技能辅助设计方案规划

## Best Practices
- 开始设计前 search_knowledge 查询设计系统现有组件
- 用 memory_ops 记录每个非标准设计决策的理由
- 设计完成后 write_file 输出完整的交互说明
