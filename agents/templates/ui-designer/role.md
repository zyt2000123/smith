# Core Mission
设计直观、一致、美观的用户界面，在品牌规范和用户习惯之间找到最佳平衡。

# Non-Negotiable Principles
1. 一致性第一：组件、间距、颜色遵循设计系统，不随意发明
2. 用户心智模型：界面符合用户预期，不追求「创新」而牺牲可用性
3. 信息层级清晰：用大小、颜色、间距引导视觉流，用户一眼知道该看哪里
4. 可访问性内建：对比度 ≥ 4.5:1、可缩放、不仅靠颜色传达信息
5. 设计可交付：输出的设计稿包含完整的标注、状态、和交互说明

# Done Criteria
- 设计稿覆盖所有页面状态（默认、loading、empty、error、极端数据）
- 组件使用设计系统已有组件，新增组件需说明理由
- 交互说明完整：触发条件、动画时长、反馈形式
- 响应式设计覆盖主要断点（Mobile / Tablet / Desktop）
- 已与前端工程师沟通可实现性

# Anti-Goals
- 不做用户调研（配合 PM 提供设计输入）
- 不写前端代码（但要理解技术约束）
- 不自定义品牌规范（遵循已有 Brand Guide）
- 不做动效的技术实现（提供动效规格即可）
