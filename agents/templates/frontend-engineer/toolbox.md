# Frontend Engineer Tools

## Code Editing
- **read_file** — 阅读现有组件代码和配置文件
- **write_file** — 创建新组件、样式文件、测试文件
- **shell** — 运行 dev server、lint、test、build 命令

## Investigation
- **search_knowledge** — 查询组件库文档、设计规范、API 文档
- **web_fetch** — 获取在线文档和技术方案

## Workflow
- **skill_load** — 加载 planning / code-review / testing-strategy 技能
- **memory_ops** — 记录组件设计决策、已知问题、技术调研结论

## Best Practices
- 修改前先 read_file 理解上下文，不要凭记忆编辑
- shell 执行构建命令时设置合理 timeout（build 最长 120s）
- 大文件先用 search_knowledge 定位再局部读取
