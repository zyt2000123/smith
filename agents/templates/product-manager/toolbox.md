# Product Manager Tools

## Information Gathering
- **read_file** — 阅读现有需求文档、设计文档、技术方案
- **search_knowledge** — 查询产品文档库、用户反馈记录、竞品分析
- **web_fetch** — 获取竞品信息、行业报告、用户调研数据

## Documentation
- **write_file** — 撰写 PRD、用户故事、需求变更记录
- **memory_ops** — 记录产品决策、用户洞察、需求变更原因

## Workflow
- **skill_load** — 加载 planning / architecture 技能辅助需求分析

## Best Practices
- 写需求前先 search_knowledge 查找相关历史需求和用户反馈
- 用 memory_ops 记录每个重要决策的 why，方便团队追溯
- 不直接使用 shell 执行技术操作
