# 开发路线图

## 已完成 (v0.1) — 基础设施与核心引擎

### common/ — 公共基础设施

- [x] 配置加载：4 层 LLM 配置合并（平台级 -> 模板级 -> Agent 级 -> 会话级）

- [x] 文件系统操作工具
- [x] 日志系统
- [x] YAML 工具

### engine/ — 自研 Agent 框架

**执行引擎**
- [x] 任务路由：3 条路由（直接回复 / 技能链 / ReAct）

- [x] 12 个门禁（Gate）
- [x] 回退机制 + `FailureLoopGuard`（`max_same=2` / `max_strategies=2`）

**LLM 客户端**
- [x] OpenAI 兼容协议
- [x] 流式输出
- [x] `tool_call` 支持
- [x] 3 次自动重试

**Prompt 组装**
- [x] 11 层 Prompt 组装（role / style / workflow / toolbox / context / memory / skills / safety / ...）

**工具系统**
- [x] 工具协议 + 注册机制
- [x] JSON Schema 自动生成

**技能系统**
- [x] 技能加载 + 注册 + 执行
- [x] 版本管理（保留 10 个版本）

**记忆系统**
- [x] agent / project 双作用域存储
- [x] Dream 整理：秘密过滤（8 regex） + 30 天剪枝 + 70% 关键词去重 + 模式提取

**用户偏好学习**
- [x] 4 维启发式检测（language / verbosity / tech_level / ...）

**插件系统**
- [x] Webhook + Polling 两种触发模式
- [x] 动态加载 handler

**安全护栏**
- [x] 24 条规则，7 大分类
